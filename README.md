## Setup
1. Run ```jasmine.py``` and complete setup with any query
2. Create ```.env``` file with:

   ```
   TELE_ID="<your_telegram_id>"
   TELE_HASH="<your_telethon_hash>"
   TELE_NUMBER="<your_telegram_number>"
   TELE_SESSION=""
   GTC_COOKIES="<getcontact.json_location>"
   ```

3. Run it again
